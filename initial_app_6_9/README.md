How to install on Android — 3 easy steps:
Step 1 — Host the HTML file for free:

On your computer, go to drop.netlify.com
Download the dharma-seekho-android.html file above
Drag and drop it onto the Netlify page
You get a free link like https://xyz.netlify.app — takes 1 minute!

Step 2 — Open on Android phone:

Open Chrome on Android
Visit your Netlify link
Tap the 3-dot menu (⋮) → "Add to Home Screen"
Tap "Add" — the app icon appears on your home screen!

Step 3 (Optional) — Make it a real APK:
If you want a proper installable Android app file:

Go to pwabuilder.com
Paste your Netlify URL
Click "Package for stores" → Android → Download APK
Install the APK on your phone (enable "Install unknown apps" in Android settings)

The game works fully offline once loaded — children can play without internet! 📱🙏
